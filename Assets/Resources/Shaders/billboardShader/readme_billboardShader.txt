Shader to correct lighting on trees' billboards [created with the built in tree creator] 
----------------------------------------------------------------------------------------


1. features
--------------------------------------------

- darkens the leaves
- brightens the bark


2. usage
--------------------------------------------

- import the package
- save, quit an restart unity in order to get the hacked shader working
- in case your billboards still look too bright, select the shaders in the project tab and refresh them. then hit "refresh tree and detail prototypes"

- The Tree folder should be in the scene folder.
